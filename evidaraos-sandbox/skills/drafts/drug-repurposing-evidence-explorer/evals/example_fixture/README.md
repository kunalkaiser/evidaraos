# Repurposing Fixture Data

These files are synthetic fixture-only examples for smoke testing. They are not scientific validation evidence and must not be used for development or marketing claims.

`literature_records.sample.json` is included only to exercise the entity extraction, relation extraction, knowledge graph, link prediction, evidence path explanation, full-text signal triage, and living evidence monitor scripts. It is synthetic fixture content, not a curated biomedical corpus.

`disease_features.sample.json` and `external_usage_signals.sample.json` are synthetic fixtures for zero-shot similarity and optional usage-signal smoke tests. They are not medical knowledge graph data, EMR data, clinical usage evidence, or validation evidence.

`method_signals.sample.json` is a synthetic fixture for method-family comparison, false-positive risk, false-negative gap, and validation-plan smoke tests. It is not a benchmark, method validation dataset, or performance estimate.
