# Payer Value Dossier Methodology Notes

Use these sources as methodological anchors when building or reviewing payer dossier outputs.

## AMCP Format Anchor

The AMCP Format for Formulary Submissions is the key US-oriented framework for clinical and economic evidence organization for healthcare decision makers.

Sources:

- AMCP Format overview: https://www.amcp.org/resource/amcp-format-5-formulary-submissions-guidance
- AMCP Format resource hub: https://www.amcp.org/resources-news/formulary-utilization-management/format-formulary-submissions
- AMCP Format 4.1 guidance page: https://www.amcp.org/resource/amcp-format-formulary-submissions-guidance-submission-pre-approval-and-post-approval-clinical-and

## Dossier Evidence Blocks

Build sections for:

- product and indication overview
- disease burden and epidemiology
- unmet need
- current treatment and comparator landscape
- clinical efficacy evidence
- safety evidence
- patient-reported outcomes and quality of life
- economic evidence
- budget impact
- cost-effectiveness evidence, if appropriate
- real-world evidence, if available
- guideline and HTA context
- access, distribution, administration, and support-program considerations
- claim traceability matrix

## Claim Controls

Each claim must be classified:

- clinical
- safety
- economic
- epidemiology
- comparator
- operational/access
- unsupported

Accepted source identifiers include:

- PMID
- DOI
- NCT ID
- HTA document URL
- FDA/EMA document URL
- public epidemiology registry URL
- model artifact ID, if generated by validated workflow

## Prohibited Behavior

- Do not predict payer decisions.
- Do not claim budget savings without sourced/model-supported evidence.
- Do not characterize unapproved products or uses as safe/effective.
- Do not leave unsupported claims in final dossier output.
