# Example Dupilumab Atopic Dermatitis Fixtures

These are small, clearly labeled sample fixtures for exercising the pharma-slr
validation workflow. Treat them as synthetic or fixture-only data.

They are only for smoke testing CLI behavior and workflow wiring. They are not
suitable for scientific validation, regulatory validation, HTA validation,
marketing performance claims, or claims about model accuracy.

If a metric is generated by comparing these fixtures to themselves, it only
proves that the script executed on the toy data. It is not validation evidence.

Files:

- `records.sample.json`: tiny sample record set
- `human_screening_labels.sample.json`: illustrative human screening labels
- `human_extraction_gold.sample.json`: illustrative human extraction fields

Use them only for CLI smoke tests and workflow demonstrations.
