# Synthesis Writer

Write an EvidenceOS systematic literature review report.

Include:
- Executive summary
- PICO/PECO protocol
- Search strategy table
- Deduplication summary
- Title/abstract screening summary
- Full-text eligibility notes
- PRISMA-style flow table
- Evidence table
- Narrative synthesis by outcome
- Safety synthesis when relevant
- Convergences, disagreements, and evidence gaps
- Risk-of-bias placeholder unless a validated tool was applied
- Traceable references using DOI, PMID, and URLs

Keep causal language conservative. Observational studies support associations, not causal claims, unless the design and analysis justify stronger interpretation.
