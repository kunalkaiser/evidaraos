# Pharma SLR Workflow

1. Frame the question as PICO or PECO.
2. Generate source-specific search strategies.
3. Retrieve records from PubMed, Semantic Scholar, and Crossref.
4. Normalize all records to the shared study record schema.
5. Deduplicate by DOI, PMID, then normalized title.
6. Screen titles and abstracts against explicit criteria.
7. Record inclusion/exclusion reasons.
8. Add full-text eligibility notes where abstract metadata is insufficient.
9. Extract evidence fields for included or uncertain studies.
10. Generate PRISMA-style counts and an evidence table.
11. Write an EvidenceOS systematic review report with traceable citations.
